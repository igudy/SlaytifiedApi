4. SlaytifiedApi.Infrastructure:

EF Core AppDbContext

Auth and Product repositories

JwtSettings config


SlaytifiedApi.Domain → Holds the entities (User, Product) and core rules about them.

SlaytifiedApi.Application → Uses DTOs and services to handle business logic (AuthService, ProductService) based on those entities.

SlaytifiedApi.Infrastructure → Handles persistence (EF Core AppDbContext, repositories) to save and load these entities from the database.

SlaytifiedApi.Api → Exposes controllers to handle HTTP requests, which call the application services.
